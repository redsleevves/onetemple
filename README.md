# OneTemple 灣廟

資策會-MMMH60 UIUX 專題發表 台灣廟宇整合平台

## 參訪網址

* 由 http://localhost/onetemple/landing_page.php 進入網站
* 可創辦帳號 以及 現有資料帳號進入 
* 帳號 : vvvvv@RVE.COM
* 密碼 : vvv

## 基本條件
* 檢視網頁 Chrome 瀏覽器
* 螢幕解析度 1080x1920 以及 手機版本 iphoneX 375x812 (由於製作時間有限 僅提供兩種版本顯示)

## 安裝
請安裝後並開啟模擬 MySQL 以及 Apache
* XAMPP
* 請先安裝XAMPP 進入 http://localhost/phpmyadmin/ 匯入資料夾當中的 onetemple.sql 資料庫
* clone下來資料夾 整包放進 C:\xampp\htdocs 當中

## 網站功能
### Landingpage
* 點擊中央可進入首頁
### 首頁
一頁式 展示網站各個功能
* 最新消息
* 探索灣廟
* 線上服務
  * 祈福點燈   
  * 求神問卜
* 聖地行旅
* 祈福商店
* 購物車
* 會員中心
  * 會員資料
  * 收藏
  * 訂單列表
### Navbar
* 網站內各個功能顯示
* 快速移動網站內
* hover時顯示英文字體
* 登入/註冊
  * 可登入現有帳號
    * 帳號
    * 密碼
  * 註冊新帳號
    * 使用者名稱
    * email 
      * email填寫欄位 有偵測是否使用過
    * 手機號碼
    * 密碼
    * 再次輸入密碼
### 最新消息
* 文章隨時間排列顯示
* 使用者可透過目錄移動
* 點擊Read More+ 可進入觀看 文章內容
* 有GoTop功能 能夠快速網頁至頂
### 探索灣廟
由於製作時間有限 彈跳介紹視窗皆為行天宮
* 使用者可透過下方箭頭 點擊移動畫面至下方探索畫面
* 使用者可透過搜尋欄位找尋廟宇
  * 搜尋欄位直接輸入搜尋 
    * 即時搜尋無須點擊
  * 可使用 全台篩選地區 下拉選擇 北 中 南 東 離島等 縮小搜尋範圍
* 熱門推薦HOT
  * 熱門廟宇點擊後 彈跳該廟宇簡介視窗
  * 右邊台灣錨點可點擊 彈跳簡介視窗
* 簡介視窗
  * 可快速導覽廟宇簡介
  * 點擊Read More 可進入 該廟宇介紹
* 廟宇介紹
由於製作時間有限 只完成行天宮廟宇介紹切版
  * 區分五個標題來介紹廟宇
    * 簡介
    * 建築
    * 歷史
    * 神明
    * 交通
* 有GoTop功能 能夠快速網頁至頂
### 線上服務
#### 祈福點燈
需登入後才能進行線上點燈服務
* 燈種介紹
  * 使用者可以透過上面的各個燈種標題 切換觀看不同燈種說明
  * 六種燈種選擇
    * 光明燈
    * 文昌燈
    * 安太歲
    * 藥師燈
    * 發財燈
    * 姻緣燈
  * 燈種資訊
    * 該燈種說明
    * 燈種費用
    * 附贈
* 線上點燈服務說明書
  * 申請流程
  * 隱私權聲明
  * 同意 與 不同意
* 線上點燈填寫表
  * 祈福者 (第一位)
    * 可快速匯入該登入會員的填寫資料
    * 必填欄位 (偵測填寫)
      * 姓名
      * 出生日期
      * 地址
      * 選擇燈種 (三重篩選)
        * 光明燈、文昌燈、安太歲、藥師燈、發財燈、姻緣燈
        * 選擇地區 (第一篩選尚未選擇時 此欄位不能選擇)
          * 北部、中部、南部、東部、離島
          * 選擇廟宇 (第一篩選尚未選擇時 此欄位不能選擇)
            * 篩選出 例如: 有提供光明燈 在北部的 行天宮
      * 新增燈種
        * 使用者能夠點多數 並選擇不同廟宇的燈種

  * 祈福者 (複數)
    * 可快速匯入該登入會員的親友填寫資料
    * 可刪除該祈福者
  * 新增祈福者
    * 新增多位祈福者
  * 加入購物車
    * 點燈填寫確認清單
      * 可讓使用者再次確認資料是否正確
#### 求神問卜
需登入後才能進行線上點燈服務
  * 使用者能夠線上求神問籤
  * 選擇神明
    * 求神步驟
  * 選擇詢問事項
  * 抽出籤詩 
    (亂數抽出籤詩)
      * 擲筊
      (亂數抽出筊)
        * 陰筊
          * 重新抽籤 
        * 笑筊
          * 重新抽籤
        * 聖筊
          * 查看籤詩
            * 加入收藏
            * 前往神明
            (該功能尚未完成)
###  聖地行旅
* 搜尋
  * 輸入搜尋字 點擊放大鏡送出
* 篩選
  * 選擇地區
    * 北部
    * 中部
    * 南部
    * 東部
    * 離島
  * 人氣推薦
  * 熱門行程
  * 最新活動
  * 價錢範圍
* 行旅商品卡
  * 收藏
  * 商品名
  * 價錢
  * 查看詳情
    * 進入商品內頁
* 排序顯示
  * 熱門
  * 價錢最高
  * 最新活動
  * 價格(低至高)
* 行旅商品內頁
  * 選擇方案及日期
    * 選擇日期
    * 選擇人數
    * 價格
      * 即時加減顯示
    * 選擇方案
    * 費用詳情
    * 總金額
      * 即時加減顯示
    * 加入收藏
    * 加入購物車
  * 行程介紹

  * 熱門推薦卡

  * 平安符
  * 聯名合作
  * 熱門商品
  * 價錢範圍

  * 價錢
  * 查看詳情

  * 熱門

  * 價格(低至高)
