# OneTemple 灣廟

台灣廟宇整合平台

## 參訪網址

* 由 http://localhost/onetemple/landing_page.php 進入網站
* 可創辦帳號 以及 現有資料帳號進入 
* 帳號 : vvvvv@RVE.COM
* 密碼 : vvv

## 基本條件
* 請先安裝XAMPP 進入 http://localhost/phpmyadmin/ 匯入資料夾當中的 onetemple.sql 資料庫
* 檢視網頁 Chrome 瀏覽器
* 螢幕解析度 1080*1920 以及 手機版本 iphoneX 375*812 (由於製作時間有限 僅提供兩種版本顯示)

## 安裝
請安裝後並開啟模擬 MySQL 以及 Apache
* XAMPP
